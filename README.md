# nestjs-microservices


# To run micro-service server
``` bash
$ cd micro-server && yarn install

# To run micro-service client in watch mode
$ yarn start:dev
```

# To run micro-service client
``` bash
$ cd micro-client && yarn install

# To run micro-service client in watch mode
$ yarn start:dev
```
